# Blog Images

博客图床仓库 - 存储博客文章中的图片和资源。

## 目录结构

```
blog-images/
├── life/       # 生活日常
├── travel/     # 旅行足迹
├── food/       # 美食记录
├── daily/      # 日常随拍
└── tech/       # 技术相关
```

## 图片引用方式

通过 jsDelivr CDN 加速访问：

```
https://cdn.jsdelivr.net/gh/israel-Liu/blog-images@main/life/photo.jpg
```

或直接使用 GitHub raw 链接：

```
https://raw.githubusercontent.com/israel-Liu/blog-images/main/life/photo.jpg
```
